WhatsApp bots run through Twilio (a messaging service).

⚠️ Note: WhatsApp requires a Twilio account. You get $15 free to test!
  1. Sign up for Twilio
  2. Set up WhatsApp Sandbox
  3. Get your credentials
  4. Copy them back to wizard
🌐

Step 1: Sign Up Twilio

Create a free Twilio account:

Sign Up for Twilio (Free)

You get $15 free credit!

📱

Step 2: Setup WhatsApp

In Twilio Console, go to WhatsApp Sandbox:

Messaging → WhatsApp → Try it out

🔑

Step 3: Get Credentials

From Twilio, copy these 3 things:

  1. Twilio Phone Number (e.g., +1 415-523-8886)
  2. Account SID
  3. Auth Token

Find these in Twilio Console → Account Settings

✅

Step 4: Paste Credentials

Go back to setup wizard and paste:

  1. Twilio SID
  2. Auth Token
  3. WhatsApp Phone Number

✓ Ready!

Go back and continue setup.
