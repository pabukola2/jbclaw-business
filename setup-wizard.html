Deploy your AI agent in 10 minutes

Welcome!

Let's set up your AI agent. This will take about 10 minutes.

We'll walk you through every step. No coding required!

What you'll do:
1. Clone the repository
2. Customize your agent
3. Add API keys
4. Start your agent
5. Done! Your agent is live
📥

Step 1: Download the Code

We'll clone the jbclaw-business repository.

📱 Open your Terminal
💻

Mac: Use Spotlight search for "Terminal"
Windows: Press Windows + R, type "cmd"
Linux: Use your terminal app

Copy this command:

git clone https://github.com/pabukola2/jbclaw-business.git my-agent && cd my-agent
What this does:
Downloads the jbclaw-business code to your computer
⚙️

Step 2: Install Everything

This will install all dependencies your agent needs.

In the same terminal, paste:

./setup.sh
What happens next:
• Downloads Python & Node.js packages
• Sets up Ollama AI models
• Configures everything automatically
Takes about 5-10 minutes
🎯

Step 3: Configure Your Agent

After setup.sh finishes, run the interactive setup.

In the same terminal, paste:

./setup-interactive.sh
This wizard will ask:
• What's your agent's name?
• How should it behave?
• Which platform? (Telegram/Discord/Slack/WhatsApp)
• Your API keys
🤖

Step 4: Name Your Agent

When the wizard asks "What's your agent's name?", you'll type something like:

Sales Bot Sales Pro Support Hero My AI Assistant Growth Partner
📝 Remember this!
Your agent will use this name when it responds to messages
✨

Step 5: Agent Personality

When the wizard asks "How should it behave?", describe your agent:

Energetic sales pro who closes deals Friendly support expert who solves problems Creative content strategist Professional business consultant
🎭 Personality matters!
This determines how your agent responds to messages
💬

Step 6: Choose Platform

When the wizard asks "Which platform?", pick one:

💡 Unsure?
Pick Telegram if you're not sure. It's easiest to set up!
🔑

Step 7: API Keys

The wizard will ask for 2 API keys. Here's where to get them:

1. OpenRouter Key (for AI models)

  1. Go to https://openrouter.ai/keys
  2. Sign in with Google
  3. Click "Create Key"
  4. Name it "jbclaw-business"
  5. Copy the key
  6. Add $10 credit (so your agent can use AI models)

2. Platform Key (Telegram/Discord/etc)

See the guide for your platform:

  • Telegram: Search @BotFather in Telegram → /newbot
  • Discord: Go to developer.discord.com → Create bot
  • Slack: Go to api.slack.com/apps → Create app
  • WhatsApp: Use Twilio (paid service)
⏱️ Takes 5 minutes
Getting API keys is usually quick. The setup wizard will guide you!
✅
