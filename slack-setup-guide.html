Let's Setup Your Slack Bot!

Simple 5-step process to get your token ready.

  1. Go to Slack API site
  2. Create new app
  3. Add bot scopes
  4. Install to workspace
  5. Copy your token
🌐

Step 1: Open Slack API

Click below to go to Slack API:

Open Slack API
📋

Step 2: Create App

Click "Create New App" → "From scratch"

Name: Your agent name
Workspace: Your Slack workspace

⚙️

Step 3: Add Scopes

Go to "OAuth & Permissions"

Add scopes:
• chat:write
• app_mentions:read

🚀

Step 4: Install & Copy Token

Click "Install to Workspace"

Copy: Bot User OAuth Token

✅

All Set!

✓ Token Ready

Go back and paste your token to continue.
